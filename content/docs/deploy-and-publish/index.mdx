---
title: Deploy & publish
description: Take a written tool from local OSS run to private deploy to a public catalog listing.
---

Three lanes for getting a tool onto Vendo: run it locally as plain open-source with your own keys, deploy it privately so only you and your team use it, or publish it to the public catalog through the `vendo-templates` repo and let any tenant install it. Each lane has its own manifest requirements, submission flow, and operational tooling (logs, debugging, updating, suspending, tearing down).
