title: Guides
description: Long-form tutorials and short copy-paste recipes for building on Vendo.
---

Guides are split into two formats. **Tutorials** are sequential, end-to-end walkthroughs — pick one when you want to build a specific kind of tool from scratch. **Recipes** are short, copy-pasteable snippets — pick one when you already know what you're building and need the exact code for a single step.
