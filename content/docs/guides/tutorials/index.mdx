Tutorials take you from an empty directory to a working tool. Each one picks a single concrete example — a Telegram bot, a CRM tool ready for the catalog, an OAuth integration — and walks through the SDK calls, `vendo.yaml` manifest, and deploy steps in order. Seed tutorials are landing in upcoming releases.
