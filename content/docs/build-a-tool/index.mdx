---
title: Build a tool
description: Author a tool that runs on Vendo — SDK, manifest, and sandbox runtime.
---

The authoring guide covers the three surfaces every Vendo tool touches: the **SDK** (tokens, connections, calling integrations through the proxy, identifying the user, billing checks, events, webhook verification), the **`vendo.yaml` manifest** (integrations, wizard inputs, env and secret declarations, healthchecks), and the **sandbox runtime** (env vars injected at boot, file system layout, gotchas).
