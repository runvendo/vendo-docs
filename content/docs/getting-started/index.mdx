## What you need

- **OSS mode:** no account required. Just set the conventional env var for each integration (e.g. `OPENAI_API_KEY`).
- **Vendo mode:** a `VENDO_API_KEY` from the [Vendo dashboard](https://vendo.run). Set it once and everything lights up.

## Next steps

After your first token call, read [Concepts: Two modes](/docs/concepts/two-modes) to understand the resolution chain, and [Concepts: Multi-tenant](/docs/concepts/multi-tenant) if you are building a SaaS product.
